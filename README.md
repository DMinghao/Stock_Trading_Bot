# Stock_Trading_Bot

## Getting Started 

### venv

- create virtual env
    ```bash 
    $ python -m venv venv
    ```

- run virtual env
    ```bash
    $ venv\Scripts\activate.bat
    ```

- exit virtual env
    ```bash 
    $ deactivate 
    ```

### Install requirements 

```bash
$ pip install -r requirements.txt
```
